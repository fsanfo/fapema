Feature FR-6 do PRD

Três painéis para validar com o PO antes da implementação em Laravel

Esta é a leitura de bancada para a equipe de desenvolvimento: cada protótipo abaixo materializa um dos painéis mandatórios definidos na seção "Homologação e governança" do PRD, usando as entidades já mapeadas no Atlas SQL (editais, chamadas, convênios e execuções). Os números são fictícios, coerentes com a granularidade descrita (mensal, trimestral, semestral, anual) — servem para validar layout, hierarquia e leitura, não para reportar.

Stack alvo: Laravel 12 + MySQL 8.0.34 ETL D+1 · camada Curated Aceite mínimo: 80% na 1ª rodada (SM-3)
